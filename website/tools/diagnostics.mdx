---
title: "Diagnostics"
description: "Always-on tools for verifying the server's health, credentials, and current safety posture."
---

These three tools are registered in every mode. They never spend money,
never need credentials, and are safe to call from any agent.

## `lighter_health`

Combined health check.

**Input** — none.

**Response**

```json
{
  "version": "0.1.0",
  "mode": "live",
  "host": "https://mainnet.zklighter.elliot.ai",
  "kit_path": "/Users/you/.lighter/lighter-agent-kit",
  "kit_python": "/Users/you/.lighter/lighter-agent-kit/.venv/bin/python",
  "audit_log": "/Users/you/.lighter/lighter-mcp/audit.jsonl",
  "system": { "code": 200, "status": "ok", "block_height": 998765 },
  "auth": {
    "auth_capable": true,
    "host": "https://mainnet.zklighter.elliot.ai",
    "account_index": 41,
    "missing": []
  }
}
```

The `system` and `auth` blocks come from `query.py system status` and
`query.py auth status` respectively, so this is the most complete
snapshot you can get without any side effects.

## `lighter_version`

Server version + active config summary + safety snapshot.

**Response**

```json
{
  "version": "0.1.0",
  "mode": "live",
  "kit_path": "/Users/you/.lighter/lighter-agent-kit",
  "host": "https://mainnet.zklighter.elliot.ai",
  "config_source": "/Users/you/.lighter/lighter-mcp/config.toml",
  "safety": {
    "mode": "live",
    "live": {
      "enabled": true,
      "allowed_symbols": ["BTC"],
      "max_order_notional_usd": 250,
      "max_daily_notional_usd": 1000,
      "max_leverage": 5
    },
    "daily_notional": { "executed_today_usd": 217.40, "remaining_usd": 782.60 }
  }
}
```

## `lighter_safety_status`

Just the safety snapshot — useful for the agent to ask "am I allowed to
do this right now?" before composing a write.

**Response**

```json
{
  "mode": "live",
  "live": {
    "enabled": true,
    "allowed_symbols": ["BTC", "ETH"],
    "max_order_notional_usd": 500,
    "max_daily_notional_usd": 2000,
    "max_leverage": 5,
    "require_confirmation": true
  },
  "funds": {
    "transfers_enabled": false,
    "withdrawals_enabled": false,
    "max_withdrawal_usd": 0
  },
  "daily_notional": { "executed_today_usd": 217.40, "remaining_usd": 1782.60 }
}
```

## Common errors

| Symptom                                                | Likely cause                                              | Fix                                                                                  |
| ------------------------------------------------------ | --------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| `kit python or script not found`                       | `kit_path` wrong, or kit's venv not built.                | Re-run the kit installer; verify `kit_path` in `config.toml`.                        |
| `auth_capable: false`, `missing: [...]`                | Credentials missing.                                      | Run the kit's onboarding, or `export` the env vars listed in `missing`.              |
| `system.code != 200`                                   | Lighter reports degraded service.                         | Wait and retry, or switch to testnet host while developing.                          |
| `safety.daily_notional.remaining_usd == 0`             | Daily cap exhausted.                                      | Wait for UTC midnight reset, or raise `max_daily_notional_usd` if intentional.       |

## Shell equivalents

```bash
# 1. Health
lighter-mcp doctor

# 2. Auth (no MCP needed)
~/.lighter/lighter-agent-kit/.venv/bin/python \
   ~/.lighter/lighter-agent-kit/scripts/query.py auth status

# 3. Tail recent failures from the audit log
jq 'select(.ok==false)' ~/.lighter/lighter-mcp/audit.jsonl | tail -n 5
```
